# Node-js-Express-js-using-Crud-Application

mkdir server

cd server 

npm init --save

npm i -g json-server

run this code in terminal 
 "server": "json-server --watch db.json --port 5000"


install these packages,

npm i express

npm i body-parser 

npm i cookie-parser 

npm i multer


To run the app use this command,

npx nodemon app.js
